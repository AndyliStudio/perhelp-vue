# perhelp

> 自由工作者平台

## 启动

``` bash
# 安装依赖
yarn install
# 启动项目
npm run dev
# 线上代码打包
npm run build
# build for production and view the bundle analyzer report
npm run build --report
# run unit tests
npm run unit
# run all tests
npm test
```
